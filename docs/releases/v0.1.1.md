# Release v0.1.1

Release date: 2026-02-28
Type: stable (`latest`)
Package: `codex-multi-auth`

## Highlights

- Fixed `codex` bin wrapper startup by lazy-loading auth runtime so clean/global
  installs avoid early module-load failures.
- Hardened installer path handling and write flow with stricter path guards plus
  atomic writes for install-time file updates.
- Improved storage root resolution to prefer directories that contain account
  data, reducing false project-root matches in nested workspaces.
- Added vendored Codex SDK/plugin dist shims so clean checkouts typecheck
  reliably without extra bootstrap steps.

## Other improvements

- Removed legacy compatibility artifacts from earlier package layouts.
- Completed codex-first naming cleanup across runtime text, prompts, and docs.
- Expanded regression coverage for wrapper concurrency and Windows
  storage-path assertions.
